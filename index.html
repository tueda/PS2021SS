参考になる（かもしれない）Web上のドキュメント

  • ゼロからのPython入門講座（Python Japan）： プログラミング入門者向けのチュートリアル。Google Colaboratoryを想定。
  • Python チュートリアル（Python Software Foundation）： Python公式サイト（本家）のチュートリアルの日本語訳。まったくのプログラミング入門者向けではない気がする。
  • ゼロから学ぶPython（渡辺宙志氏）： Web上に公開されている講義録。Google Colaboratoryを想定。最近、本になった。
  • プログラミング演習 Python 2019（喜多一氏）： PDFファイルとして公開されている講義録。IDLEを想定。
  • Introduction to Computer Science and Programming in Python（MIT OpenCourseWare）： プログラミング入門者向け（英語）。
  • Python Programming And Numerical Methods: A Guide For Engineers And Scientists（Qingkai Kong氏、Timmy Siauw氏、Alexandre Bayen氏）： プログラミング＆数値計算の入門者向けの本のネット公開版（英語）。
